üìã How to Install This Booking Engine

Step 1: No Download Required!

The script is hosted on GitHub Pages - no need to download files.

Step 2: Add One Line to Your HTML

Add this script tag before the closing </body> tag:

<script src="https://samuelgfenton.github.io/dudaTest/bookNow/siteminder_script.js" 
        data-booking-url="https://your-booking-engine-url.com"></script>

Step 3: Customize (Optional)

Add these attributes to customize the button:

<script src="https://samuelgfenton.github.io/dudaTest/bookNow/siteminder_script.js" 
        data-booking-url="https://your-booking-engine-url.com"
        data-button-text="Reserve Now"
        data-button-color="#e53e3e"
        data-header-color="#2d3748"></script>

Configuration Options:

  • data-booking-url: Your Siteminder booking engine URL (required)
  • data-button-text: Button text (optional, defaults to "Book Now")
  • data-button-color: Button background color (optional)
  • data-header-color: Popup header color (optional)
